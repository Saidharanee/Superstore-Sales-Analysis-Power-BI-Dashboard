# Superstore-Sales-Analysis-Power-BI-Dashboard

## Project Overview
This Power BI project analyzes the Superstore dataset to provide insights into sales, profit, and regional performance. The dashboard helps stakeholders identify top-performing categories, loss-making segments, and sales trends.

## Objectives
- Analyze total sales, profit, and quantity
- Identify top and bottom performing products
- Region-wise and segment-wise performance
- Profitability analysis
- Sales trend over time

## 📂 Dataset
- Source: Superstore dataset
- Contains: Orders, Sales, Profit, Category, Region, Ship Mode

## 📊 Key Dashboard Features
- KPI Cards → Sales, Profit, Quantity
- Region-wise Sales Analysis
- Category & Sub-category performance
- Monthly Sales Trend
- Top 10 Profitable Products
- Loss-making segments identification

## 🛠️ Tools & Technologies
- Power BI
- Power Query (Data Cleaning & Transformation)
- DAX (Calculated measures)

## 📈 Key Insights
- Technology category generated the highest profit
- Certain sub-categories show consistent losses
- West region contributed the highest sales
- Sales peak observed during year-end season
